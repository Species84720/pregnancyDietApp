# Phase 0: Project Foundation

## Goal
Create or verify the Android project foundation for the Pregnancy Diet Tracker app.

## Scope
1. Set up a modern Android project using Kotlin.
2. Use Jetpack Compose for UI.
3. Set up a clean package structure:
   - auth
   - onboarding
   - pregnancy
   - symptoms
   - supplements
   - nutrition
   - meals
   - ai
   - data
   - firebase
   - ui
   - common
4. Add Gradle dependencies needed for:
   - Firebase Auth
   - Firebase Firestore
   - Firebase Cloud Messaging placeholder if needed later
   - Jetpack Compose
   - Navigation Compose
   - Kotlin coroutines
   - ViewModel
5. Add basic app theme.
6. Add placeholder navigation routes.
7. Add README setup instructions if missing.
8. Ensure the project builds.

## Out of scope
- Do not implement authentication yet.
- Do not implement real pregnancy forms yet.

## Acceptance criteria
- Project opens in Android Studio.
- App builds successfully.
- Compose navigation shell exists.
- Package structure is present.
- No secrets are committed.

## Required commit message
`Phase 0: Set up Android project foundation`
