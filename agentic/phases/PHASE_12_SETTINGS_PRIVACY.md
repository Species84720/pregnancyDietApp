# Phase 12: Settings, Privacy, and Account Deletion

## Goal
Add settings, privacy controls, and account deletion flow.

## Scope
1. Add Settings screen.
2. Include:
   - profile details
   - pregnancy profile edit
   - allergies
   - dietary restrictions
   - medical conditions
   - notification settings
   - data export entry point
   - sign out
   - delete account
3. Implement account deletion flow.
4. Delete or mark for deletion user-owned Firestore data according to project rules.
5. Add confirmation dialogs for destructive actions.
6. Add privacy and medical disclaimer screens.
7. Ensure Firebase Security Rules are documented.
8. Make sure app handles signed-out state correctly.

## Acceptance criteria
- Settings screen contains all required entries.
- User can edit profile-related settings.
- Sign out works.
- Account deletion has confirmation and user-scoped data handling.
- Privacy and medical disclaimer screens exist.

## Required commit message
`Phase 12: Add settings privacy and account deletion`
