# Phase 10: Reminders and Notifications

## Goal
Add gentle reminders for supplements, meals, and symptom check-ins.

## Scope
1. Add reminder settings for supplements.
2. Add optional meal logging reminders.
3. Add optional daily symptom check-in reminder.
4. Use Android notification best practices.
5. Support notification permission flow.
6. Store reminder preferences in Firestore or local settings as appropriate.
7. Allow user to disable reminders.
8. Do not spam users.
9. Add notification copy that is gentle and non-alarming.
10. Add tests where possible.

## Acceptance criteria
- User can enable/disable reminder types.
- Supplement reminders can be scheduled.
- Notification permission flow works.
- Copy is gentle and not alarming.

## Required commit message
`Phase 10: Add reminders and notifications`
